# W-Ryde: Write your Ride

W-Ryde allows you to read vehicle experiences from the actual owner and also contribute your own. Made with NextJS, TypeScript, ASP.NET Core Web API, Microsoft SQL Server, Redis and ElasticSearch.

## Run Locally

Clone the project

```bash
  git clone https://github.com/iapheus/wryde.git
```

Go to the project directory

```bash
  cd wryde
```

Install dependencies

```bash
  npm install
```

Next, you need to run the backend.

Clone the project

```bash
  git clone https://github.com/iapheus/wryde-backend.git
```

Go to the project directory

```bash
  cd wryde
```

Run the backend

```bash
  dotnet run
```

Afterwards, run the frontend

```bash
  npm run dev
```
